Vim insert

NORMAL 模式下：

  • i 在光标处进入插入模式
  • I 光标移至当前行首位，进入插入模式
  • a 光标后移一位，进入插入模式
  • A 光标移至当前行末位，进入插入模式
  • o 在当前行下添加一个空行，光标在空行开始，进入插入模式
  • O 在当前行上添加一个空行，光标在空行开始，进入插入模式

复制，剪切和粘贴

复制

NORMAL 模式下：

  • y 在使用 v 模式选定区域时，复制选定块
  • yy 复制整行
  • nyy / yny 从当前光标向下复制 n 行
  • y^ 复制当前行光标到首位的内容
  • y$ 复制当前行光标到末位的内容
  • yw 复制当前 word
  • yG 复制当前光标到文件末位的内容
  • nyG / ynG 将光标移至第 n 行，再复制到文件末位的内容

剪切

NORMAL 模式下：

  • d 在使用 v 模式选定区域时，剪切选定块
  • dd 剪切整行
  • ndd / dnd 从当前光标向下剪切 n 行
  • d^ 剪切当前行光标到首位的内容
  • d$ 剪切当前行光标到末位的内容
  • dw 剪切当前 word
  • dG 剪切当前光标到文件末位的内容
  • ndG / dnG 将光标移至第 n 行，再剪切到文件末位的内容

粘贴

NORMAL 模式下：

  • P 粘贴至光标后（下）

  • p 粘贴至光标前（上）

    整行粘贴时在光标 上 / 下，非整行粘贴时在光标 前 / 后

命令前面加数字表示重复的次数，加字母表示使用的缓冲区名称。使用英文句号 . 可以重复上一个命令。

撤销和重做

NORMAL 模式下：

  • u 撤销
  • ctrl + r 重做

NORMAL 模式其他

  • x 删除光标处的字符
  • r 按下之后再输入一个字符，替换掉光标处的字符
  • g 移动当前行到第一行
  • G 移动当前行到最后一行
  • ng 移动当前行到第 n 行
    • 命令模式输入 set nu 显示行数
  • ^ 光标移至当前行首位
  • $ 光标移至当前行末位
