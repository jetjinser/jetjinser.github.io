用户和用户组管理

发表于 2020-04-30

用户

新建用户

useradd {用户名}

没有指定组时，会自动创建一个同用户名的组

查看用户

id {用户名}

​ 不提供用户名时查看当前用户

root 用户的家目录是 /root

其他用户的家目录是 /home/用户名

修改密码

passwd [用户名]

没有指定用户时修改当前用户的密码

删除用户

userdel {用户名} [-r]

加 -r 表示将配置文件也删除，包括 /home/用户名 文件夹

配置用户

  • usermod
    • -d {目录} {用户名} 设定家目录
    • -g {组名} {用户名} 修改用户到用户组
    • …

chage

…

用户组

新建用户组

groupadd {组名}

修改用户到用户组

usermod -g {组名} {用户名}

在创建用户时添加到组

useradd -g {组名} {用户名}

切换用户

  • su - {用户名} 切换用户

  • su {用户名} 不完全切换，目录没有移动到用户的家目录

    普通用户之间切换需要密码, root 切换到普通用户不需要密码
